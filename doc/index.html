Welcome to sutil’s documentation!¶

SoldAI utils (sutil) is a set of tools to help ML scientist and programmers to build information systems for classification. It let’s you define your very own handcrafted models, embed sklearn models or create keras and pytorch models and test it faster. It also includes a set of utilities to work with text perform cleansing and pre processing.
